# プロジェクト概要

TODOアプリケーション

## プロジェクトタイプ
学習・実践プロジェクト（ドメイン駆動設計の習得）

## 実装範囲
個人利用を想定したタスク管理アプリケーション

## 主要機能
- **タスク管理**: タスクの作成、閲覧、編集、削除、ステータス変更
- **リスト管理**: タスクリストの作成、編集、削除、タスクの分類
- **フィルタリング・ソート**: ステータスによるフィルタリング、期日によるソート
- **タスク移動**: タスクの別リストへの移動

## アーキテクチャ方針
- ドメイン駆動設計（DDD）の実践
- ポート&アダプターアーキテクチャによる依存関係の制御
- テスタビリティと保守性を重視した設計
- SOLID原則の適用
